# quantic

A lightweight Python number formatter that converts large numbers into readable suffix formats (like `10k`, `1.2M`, `3.4B`).

Simple. Fast. Minimal.

---

## 🚀 Features

- Converts large numbers into short suffix form
- Easy-to-use `Run()` function
- Supports common incremental suffixes (k, M, B, T, etc.)
- Lightweight with no dependencies
- Perfect for games, UI displays, and simulators
