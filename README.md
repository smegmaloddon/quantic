# quantic

A minimal Python package scaffold including utilities, module initialization, and CLI entrypoint support.

## Install

```bash
python -m pip install .
```

## Usage

Run the package as a module:

```bash
python -m quantic
```

Use the CLI entrypoint:

```bash
quantic
```

## Features

- `quantic.utils` helper functions
- `__main__` entrypoint for `python -m quantic`
- package metadata via `pyproject.toml`
